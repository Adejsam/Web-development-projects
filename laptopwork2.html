Database

A Datables is a structued and organised collection of data. This data can be collected in tables, heirachy or network. Data collected in tables are called relational Database while others are called Non-relational database

  • Steps in Developing a Database

    1. Collect or Gather Business Requirement
    2. Develop a Entity Relationship Model
    3. Develop a conceptual model
    4. Develop a Physical Model
    5. Implement the Database
    6. Test the Database
    7. Documentation

  • DBMS (Database Management System)

    This are various systems or Application that can be used to Manage a database. This include but not limited to

    1. MySQL
    2. PostgreSQL
    3. Oracle
